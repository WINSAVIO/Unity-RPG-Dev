## üìñ About The Project

This project serves as a comprehensive study of RPG mechanics. It features a responsive combat system where the player can unlock various abilities through a skill tree, manage equipment via an inventory system, and fight enemies driven by intelligent AI.

### Key Technical Features
* **Finite State Machine (FSM):** Player and enemy behaviors (Idle, Move, Attack, Stun) are managed through a robust state machine pattern, ensuring clean and maintainable code.
* **Skill System:** A fully functional Unlockable Skill Tree that dynamically updates player abilities (e.g., Sword Throw, Dash, Ultimate).
* **Inventory & Stats:** Backend systems for managing items, equipment stats, and player attributes.
* **Parallax & VFX:** Polished visuals with parallax background layers and particle effects for combat impact.

## üõ†Ô∏è How to Run (Installation)

Since this project utilizes Unity packages, the source files are compressed to ensure easy distribution.

1.  **Download:** Download the `RPG.zip` file from this repository.
2.  **Extract:** Unzip the contents of `RPG.zip` to a folder on your computer.
3.  **Open Unity Hub:** Launch Unity Hub.
4.  **Add Project:** Click the **"Add"** button (or "Add project from disk").
5.  **Select Folder:** Navigate to the extracted folder containing the `Assets` and `ProjectSettings` directories and select it.
6.  **Launch:** Click on the project name in Unity Hub to open the editor.
7.  **Play:** Open the `MainScene` (usually in `Assets/Scenes`) and press the **Play** button at the top.
